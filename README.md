# MySpa Germany - Interactive Business Dashboard

## Project Overview
Interactive business intelligence dashboard analyzing 50,000 spa booking records 
across 5 German cities. Built to simulate real-world data analytics workflows 
and support data-driven business decisions.

## Dashboard Features
- Total Profit by City
- Revenue Trend by Month
- Revenue Distribution by Service Type
- Average Customer Satisfaction by City

## Tools & Libraries
- Python
- Pandas
- Plotly — interactive visualizations
- Faker — mock data generation

## Key Insights
- Berlin leads in customer satisfaction
- May generates the highest monthly revenue
- All 5 branches maintain competitive profit margins

## Files
- `myspa_dashboard_interactive.ipynb` - Full analysis notebook
- `complete_dashboard.html` - Interactive dashboard
- `city_profit.html` - Profit by city chart
- `monthly_revenue.html` - Revenue trend chart
- `service_revenue.html` - Service distribution chart
- `satisfaction.html` - Customer satisfaction chart
